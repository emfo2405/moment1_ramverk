Bootstrap

Information om ramverket

Bootstrap är, likt Tailwind, ett CSS-ramverk som gör att arbetet med designen av HTML-element blir enklare. I Bootstrap används klasser som sätts på HTML-elementen och dessa innehåller då olika kod för CSS-regler. Med ramverket kan man bland annat ändra saker som färger och padding. Bootstrap har även klasser för att designa större komponenter och har därmed mindre flexibilitet i designen. Med de inbyggda designvalen för komponenter kan man exempelvis enkelt göra stilval för en tabell med några få klasser. Bootstrap har också ett antal förvalda färgval vilket ofta gör att webbsidor designade med Bootstrap får ett liknande utseende.

Utvärdering

Jag tycker att arbetet med Bootstrap har gått bra. Installationen tycker jag var lite svårare än för Tailwind men det gick ändå hyfsat lätt. Efter att ha arbetat med Tailwinds klasser blev det ganska naturligt hur man använde klasser även med Bootstrap. Även här var det skönt att det fanns en cheat sheet så att man enkelt kunde hitta rätt klasser för ändamålet. Jag tycker att stilvalen för hela komponenter, så som för tabeller, gjorde att det gick väldigt fort att jobba med Bootstrap men med lite mindre anpassningsmöjligheter om man endast ska hålla sig till de satta klasserna. Sedan la jag till lite CSS själv också för att skapa ramar runt element. Fördelarna jag ser med Bootstrap är att det går snabbt och enkelt att skapa design för hemsidor. Det är även praktiskt att kunna designa större element med väldigt få klasser och väldigt enkelt att kunna anpassa element till olika skärmstorlekar. Nackdelen är att det är lite färre valmöjligheter för design och färg. Även att man behöver upprepa en del kod i filen och att det blir lite rörigt efter ett tag med alla uplacerade klasser.
