Publiceringsplattform

För den här uppgiften har jag valt att publicera min webbplats på Microsoft Azure. Anledningen till valet var främst att jag inte har använt plattformen innan men också för att det är en populär publiceringsplattform. Den här plattformen har både gratis- och betaltjänster. Dock avslutas gratisperioden efter ett tag. Plattformen stödjer statiska webbplatser men även backend-baserad utveckling. Microsoft Azure har stöd för ett flertal olika programmeringsspråk och de är bland annat C#, Java, JavaScript, PowerShell, Python och Typescript. Eftersom Azure är en väldigt populär publiceringsplattform finns det även mycket information om det och sätt att få hjälp om man stöter på problem.

Man kan ladda upp webbplatser med Azure på några olika sätt men det jag gjorde var att ladda upp min skapade webbplats till GitHub och sedan koppla denna till Azure och mitt konto där. Sedan gör man val för sin publikation och sedan skapas den. Om man kopplar den till sitt GitHub-repo uppdateras den även varje gång man pushar nya ändringar till det. Fördelen med Azure tycker jag är att det är ett enkelt sätt att ladda upp och skapa sin webbplats. Det är också enkelt att hitta svar på de frågor som kan uppstå under processen. Nackdelen är att man inte kan använda tjänsten gratis för alltid utan tillslut måste man betala för att ha tillgång till tjänsten.
