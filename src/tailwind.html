Tailwind

Information om ramverket

Tailwind är ett CSS-ramverk som är skapat för att underlätta utvecklingen av webbplatser. Ramverket använder klasser med designval kopplade till sig som sätts på olika HTML-element. Detta gör att man kan skapa designen av hemsidan från HTML-filen och även att den är väldigt anpassningsbar till användarens behov. Tailwind består av ett flertal olika klasser för saker som färg, storlek, margins med mera så att man kan forma elementen som man vill. Man kan även lägga till egna färger eller typsnitt till Tailwind så att man kan formatera alla element precis som man vill.


Klicka här för att komma till tailwinds sida.

Utvärdering

Jag tycker att arbetet med Tailwind har gått bra. Det var enkelt att installera via npm och även enkelt att sätta sig in i. Jag tycker det blir enkelt att lägga till klasser till HTML-elementen men det blir också lite stökigt om man vill ha många olika designval på samma element.
