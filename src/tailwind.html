Tailwind

Information om ramverket

Tailwind är ett CSS-ramverk som är skapat för att underlätta utvecklingen av webbplatser. Ramverket använder klasser med designval kopplade till sig som sätts på olika HTML-element. Detta gör att man kan skapa designen av hemsidan från HTML-filen och även att den är väldigt anpassningsbar till användarens behov. Tailwind består av ett flertal olika klasser för saker som färg, storlek, margins med mera så att man kan forma elementen som man vill. Man kan även lägga till egna färger eller typsnitt till Tailwind så att man kan formatera alla element precis som man vill.

Utvärdering

Arbetet med Tailwind har gått bra och jag tyckte det var enkelt att både installera via npm och att sätta sig in i. Det blir ett väldigt tydligt system när man lägger till designval i HTML-koden med olika klasser för de val man vill ha. Jag gillar också att det finns cheat sheets för de olika klasserna så att utvecklingen och arbetet går smidigare. De fördelar jag ser med att arbeta med Tailwind som ramverk är att man kan välja designval i HTML-koden och allt är samlat på en plats. Det blir också lätt att anpassa element till olika skärmstorlekar direkt. Jag gillar även att man kan lägga till egna färger m.m. för att anpassa stilval ännu mer till ens egna preferenser. Nackdelen tycker jag är att det blir väldigt mycket kod på en plats och att det blir svårare att hitta specifika delar när man ska gå tillbaka i sin kod och ändra saker. Jag tycker även att det blir en nackdel att man måste upprepa samma koddel för de element som ska ha samma stil som ett annat element.
